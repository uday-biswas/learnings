roadmap for devops

1. programming language - python , golang
2. server administration - linux
3. networking and security - tcp/ip protocols
4. servers 
    a. web server - apache, nginx
    b. cache server - redis, memcached
    c. database server - mysql, postgresql, mongodb

5. infrastructure as code 
    a. configuration management - ansible, puppet, chef
    b. containerization - docker
    c. container orchestration - kubernetes, docker swarm
    d. infrastructure provisioning - terraform, cloudformation or cloud providers such as google deployment manager, aws cloudformation, azure resource manager
6. ci/cd - jenkins, circleci, travis ci, github actions
7. monitoring and logging - prometheus, grafana, thanos
8. cloud - aws, gcp, azure
